Regardless of the topic you would like to work on, you will need to get yourself familiar with the GitHub flow. Please refer to the OSCD How-To to get yourself fully prepared for the Sprint.

How to pick up a task

If you would like to assign yourself to some of the tasks listed in the GitHub issue, you should comment on the issue with a specific task (usually it has a number) you are going to solve. This way, the other participants will see that you will work on a particular task so they will do something else and not intersect with you.
